# Yesilkoy Approach, Istanbul TMA 

So far the biggest project that I made, File is going through renovation so most of things are unfinished.

- Currently;
    - LTFM, LTFJ and LTBA aerodromes included.
    - Departures for all airports and for all configurations are available.
    - Triple runway operations (TRO) for LTFM and simultaneous paralel departures for LTFJ supported.
    - All STARs are included for LTFM but due technical impossibilities, aircraft will follow pre selected STAR at the start.
    - It is possible for select another appproaches by directing aircrafts to specific fix.
    - North configuration (36/35R/34L-06L/06R-05) almost all approach fixs for LTFM are included. Only few selected fixs for LTBA and LTFJ are available.
    - South configuration (18/17L/16R-24R/24L-23) STARs for LTFM available but most fixes are not added yet. For LTBA and LTFJ STARs not added yet.
    - Bandirma TMA available. P-D-R areas not added.
    
 - In future;
    - All airports in boundary (LTFM/LTFJ/LTBA/LTBU/LTBQ/LTBR/LTBW/LTBG/LTBP) will be added.
    - All STARs and SIDs for all configurations will be available.
    - New SIDs at upcoming AIRAC will be added. 
    - Realistic aircraft fleets will be addded.
    - VFR traffic will be added.
    - Helicopter traffic for off-shore platforms in Black Sea will be added.
