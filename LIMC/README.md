# Milano Terminal Maneuver Area 

1.  Only contains Milano Malpensa (LIMC) other aerodromes in TMA (LIML/LIME/LSZA/LIMN) are not added yet.
  
2.  For LIMC it includes:
- All runways 35L/35R and 17L/17R included. 
- All STARs.
- For SIDs, fixed initial climb procedures but all transition are included.
  
3.  Includes TMA areas but P-D-R and CTR areas are not added yet.
